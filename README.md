# Apuntes de la materia de Minería de Datos

Apuntes de [Data Mining](https://mariosky.github.io/DataMining/)


Para generar los apuntes en tu máquina:

1. Clona el repositorio
2. Crea un ambiente virtual de Python
3. Instala los requerimentos en `requirements.txt`
4. Corre el comando de Sphinx para generar el libro en `html` 
5. Abre el archivo generado `index.html`

Ejemplo:

```bash
gh repo clone mariosky/DataMining
cd DataMining
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
make html
open build/html/index.html
```
